## How it works

Seccion9 runs a WireGuard gateway on your VPS. Clients tunnel through it to reach internal resources, with all traffic encrypted end-to-end. A FastAPI backend and React frontend let you manage peers without touching the command line.

<Steps>
  <Step title="Install the server">
    Run `sudo bash server/install-server.sh` on your Ubuntu 22.04 VPS to install WireGuard, generate server keys, configure UFW, and start the `wg0` interface.
  </Step>
  <Step title="Deploy the management panel">
    Configure `panel/docker-compose.yml` with your credentials and public IP, then run `sudo bash panel/deploy.sh`. The dashboard is accessible at `http://YOUR_IP:8443`.
  </Step>
  <Step title="Add a client">
    From the panel, click **Add client**, enter a name, and download the generated `.conf` file or scan the QR code on mobile.
  </Step>
  <Step title="Connect">
    Import the `.conf` into the WireGuard app on Windows, Linux, macOS, or iOS/Android and toggle the tunnel on.
  </Step>
</Steps>

## Choose your protocol

| Client firewall | Recommended protocol |
|---|---|
| pfSense / OPNsense / MikroTik | WireGuard (UDP 51820) |
| Cisco ASA / Fortinet / SonicWall | OpenVPN (TCP 443) |
| No corporate firewall (cloud) | WireGuard (UDP 51820) |

<Note>
  Many cloud providers (Ionos, OVH, Hetzner) have an **external firewall** independent of UFW. You must open UDP 51820 and TCP 8443 in your provider's network policy before clients can connect. See [Firewall configuration](/server/firewall) for details.
</Note>
